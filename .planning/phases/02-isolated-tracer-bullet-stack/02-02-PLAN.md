<threat_model>
**ASVS L1 — block on high.**
- **T-02-04 (Info disclosure / credential handling, HIGH if mishandled):** auth must reach the instance without leaking. Mitigation: mount `~/.claude/.credentials.json` **read-only**; the generated `.claude.json` stub carries identity/onboarding fields but **NO token**; never copy the credential into the profile or an image layer (env/ro-mount only, §7/§16). Disposition: mitigate.
- **T-02-05 (Tampering, host `.claude.json`):** rw-mounting the host whole-file blob corrupts host state (Pitfall 1). Mitigation: isolated GENERATES a stub and never mounts the host file (MNT-03 carried forward). Disposition: mitigate.
- **T-02-06 (Spoofing/DoS, MCP endpoint):** hatago listens on the pod netns. Mitigation: the endpoint is `localhost:3535` inside the shared pod netns (not host-exposed); the egress firewall (§4a) still applies to the harness container. Disposition: mitigate.
</threat_model>
