## Phase Goal

**As a** stack author, **I want to** provide snyk and Socket.dev tokens so `harnessed build` runs them for deeper supply-chain signals, and get a clean warn-and-skip when I don't, **so that** my build gains optional scanner depth without ever prompting or regressing the credential-free default gate.

## Phase User Story (MVP vertical slice — SEC-02)

This plan delivers the foundational tools-image layer (Node + varlock + op + snyk + socket) AND the first vertical slice: token-gated scanners that run during `harnessed build`. Every subsequent Phase 5 plan (05-02 secrets, 05-03 timer) depends on this image foundation.
